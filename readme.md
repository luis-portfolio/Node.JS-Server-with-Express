# Small Test Enviroment NodeJS Server with Express

## Routes dynamically on folder /app/routes

### user resource

> GET /user

> GET /user/:id

> POST /user

> PUT /user/:id

> DELETE /user/:id

### user fields 

> id, name, lastName, age, birthday, email

### file resource

> POST /file/save

### Installing uses yarn

```terminal
yarn 
```

### Installing uses npm

```terminal
npm i 
```

### Starting modes with yarn

```terminal
yarn dev
```

```terminal
yarn debug
```

```terminal
yarn start
```

### Starting modes with npm

```terminal
npm run dev
```

```terminal
npm run debug
```

```terminal
npm start
```
